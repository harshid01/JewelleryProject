# 📖 Project Overview

The **Jewellery E-Commerce Website** is a complete online shopping platform designed to simplify the process of browsing, purchasing, and managing jewellery products. The system delivers a secure, responsive, and user-friendly shopping experience while providing administrators with comprehensive tools to manage every aspect of the online store.

Customers can register, browse products, search by category, manage shopping carts, place orders, and maintain their profiles. Administrators have access to a dedicated dashboard for managing products, categories, customers, orders, inventory, banners, and website content.

This project demonstrates practical implementation of **Full Stack Web Development**, **Database Management**, **Authentication**, **CRUD Operations**, and **Responsive Web Design**.
